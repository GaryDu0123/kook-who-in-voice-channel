# kook-who-in-voice-channel

# 安装

## 将项目源代码下载或者clone到本地

```bash
git clone https://github.com/GaryDu0123/kook-who-in-channel.git
```

## 将解压出来的文件夹放到`moudules`目录下

期望结构: 

```text
modules/
  ...
  ├── kook-who-in-voice-channel/                            
      ├── __init__.py                          
      ├── assets                               
      │   └── voice.png                        
      ├── config.json
      ├── kook_channel_voice.py
      ├── msyh.ttc
      └── msyhbd.ttc
  ...
```
